Tutorial Desain Jersey di CorelDraw

Apa Itu CorelDraw?

CorelDraw adalah sebuah software untuk desain grafis yang dikembangkan oleh Corel Corporation. Software ini memungkinkan kamu untuk membuat desain gambar, ilustrasi, dan juga lukisan. Selain itu, CorelDraw juga dapat digunakan untuk membuat desain jersey.

Cara Membuat Desain Jersey di CorelDraw

Langkah 1: Buka Software CorelDraw

Pertama, buka software CorelDraw. Setelah itu, pilih New untuk membuat file baru. Pada kolom Width dan Height, tulis ukuran desain jersey yang kamu inginkan. Lalu, tekan OK.

Langkah 2: Tambahkan Bentuk dan Coretan

Kedua, tambahkan bentuk dan coretan pada desain jersey. Caranya, pilih Toolbox. Lalu, pilih shape yang kamu inginkan. Sebagai contoh, pilih elips. Selanjutnya, drag dan drop elips tersebut ke file CorelDraw kamu. Ulangi cara ini sampai kamu memiliki desain yang kamu inginkan.

Langkah 3: Tambahkan Warna

Ketiga, tambahkan warna pada desain jersey. Caranya, pilih Toolbox. Lalu, pilih Fill Tool. Sekarang, klik pada bagian desain jersey yang ingin kamu beri warna. Lalu, pilih warna yang kamu inginkan. Ulangi cara ini sampai kamu memiliki warna yang kamu inginkan.

Langkah 4: Tambahkan Teks

Keempat, tambahkan teks pada desain jersey. Caranya, pilih Toolbox. Lalu, pilih Text Tool. Sekarang, klik pada bagian desain jersey yang ingin kamu beri teks. Lalu, ketikkan teks yang kamu inginkan. Ulangi cara ini sampai kamu memiliki teks yang kamu inginkan.

Langkah 5: Download dan Cetak

Kelima, download dan cetak desain jersey. Caranya, pilih File. Lalu, pilih Save As. Pilih format file yang kamu inginkan. Setelah itu, pilih file lokasi yang kamu inginkan. Sekarang, kamu sudah dapat mencetak desain jersey yang kamu buat.

Kesimpulan

Itulah cara membuat desain jersey di CorelDraw. CorelDraw memang sangat mudah digunakan untuk membuat desain jersey. Jadi, jangan ragu untuk mencoba membuat desain jersey di CorelDraw.
