Tutorial Edit Foto Menggunakan Corel Draw X7

Perkenalan Corel Draw X7

Corel Draw X7 adalah salah satu software editing foto yang banyak digunakan oleh para desainer. Software ini memberikan berbagai macam fasilitas dalam mengedit gambar sehingga menghasilkan hasil yang profesional.

Menginstal Corel Draw X7

Pertama yang harus dilakukan adalah menginstal Corel Draw X7. Caranya cukup mudah, Anda bisa mendownload file Corel Draw X7 pada link berikut: https://www.coreldraw.com/en/pages/downloads/. Setelah file terdownload, Anda bisa menjalankan file tersebut untuk melakukan instalasi.

Membuka Gambar di Corel Draw X7

Setelah Corel Draw X7 berhasil diinstal, Anda bisa langsung membuka gambar yang akan diedit. Caranya cukup mudah, Anda tinggal klik menu File > Open. Lalu Anda akan diarahkan ke folder dimana gambar tersebut disimpan. Pilih gambar yang akan diedit, lalu klik Open.

Mengubah Ukuran Gambar di Corel Draw X7

Setelah gambar berhasil dibuka, Anda bisa mengubah ukuran gambar tersebut sesuai keinginan. Caranya cukup mudah, Anda tinggal klik menu Image > Resize. Lalu Anda akan diberikan pilihan untuk mengubah ukuran gambar. Pilih ukuran yang Anda inginkan, lalu klik OK.

Mengubah Warna Gambar di Corel Draw X7

Selain mengubah ukuran gambar, Anda juga bisa mengubah warna gambar menggunakan Corel Draw X7. Caranya cukup mudah, Anda tinggal klik menu Image > Color. Lalu Anda akan diberikan pilihan untuk mengubah warna gambar. Pilih warna yang Anda inginkan, lalu klik OK.

Menambahkan Objek di Corel Draw X7

Selain mengubah warna gambar, Anda juga bisa menambahkan objek ke dalam gambar. Caranya cukup mudah, Anda tinggal klik menu Object > Insert. Lalu Anda akan diberikan pilihan untuk menambahkan objek. Pilih objek yang Anda inginkan, lalu klik OK.

Menyimpan Gambar di Corel Draw X7

Setelah selesai mengedit gambar, Anda bisa langsung menyimpannya. Caranya cukup mudah, Anda tinggal klik menu File > Save. Lalu Anda akan diarahkan ke folder dimana gambar akan disimpan. Pilih lokasi penyimpanan, lalu klik Save.

Mencetak Gambar di Corel Draw X7

Selain menyimpannya, Anda juga bisa mencetak gambar yang telah diedit. Caranya cukup mudah, Anda tinggal klik menu File > Print. Lalu Anda akan diberikan pilihan untuk mencetak gambar. Pilih jenis kertas yang Anda inginkan, lalu klik Print.

Menutup Gambar di Corel Draw X7

Setelah selesai mencetak gambar, Anda bisa langsung menutup gambar tersebut. Caranya cukup mudah, Anda tinggal klik menu File > Close. Lalu Anda akan diberikan pilihan untuk menutup gambar. Pilih Close, lalu klik OK.

Menutup Corel Draw X7

Setelah selesai menutup gambar, Anda bisa langsung menutup Corel Draw X7. Caranya cukup mudah, Anda tinggal klik menu File > Exit. Lalu Anda akan diberikan pilihan untuk menutup Corel Draw X7. Pilih Exit, lalu klik OK.

Kesimpulan

Itulah tutorial cara edit foto menggunakan Corel Draw X7. Dengan menggunakan Corel Draw X7, Anda bisa mengedit foto dengan mudah dan hasil yang profesional. Selamat mencoba!
