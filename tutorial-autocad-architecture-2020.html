Tutorial Autocad Architecture 2020: Belajar Dasar Autocad dengan Mudah

Apa itu Autocad Architecture 2020?

Autocad Architecture 2020 adalah aplikasi desain dan drafting yang memungkinkan Anda untuk menggambar dan membuat desain 2D dan 3D untuk arsitektur. Autocad Architecture 2020 dapat digunakan untuk merancang, menggambar, dan mengedit detail arsitektur. Aplikasi ini memiliki berbagai fitur dan alat yang dapat membantu Anda dalam menciptakan desain yang Anda inginkan dengan mudah.

Keuntungan Menggunakan Autocad Architecture 2020

Autocad Architecture 2020 memiliki berbagai macam fitur yang berguna untuk membantu Anda menciptakan desain arsitektur yang Anda inginkan. Fitur-fitur tersebut meliputi:

  • Kompatibilitas yang tinggi dengan format file CAD lainnya.
  • Alat dan fitur yang sangat kuat untuk menciptakan desain arsitektur yang kompleks.
  • Penempatan komponen yang mudah.
  • Kelengkapan dan keteraturan desain.
  • Dokumentasi yang mudah dari desain Anda.
  • Ketepatan dan kecepatan.
  • Kompatibilitas dengan berbagai jenis file.
  • Peningkatan produktivitas.

Cara Install Autocad Architecture 2020

Berikut adalah langkah-langkah untuk menginstal Autocad Architecture 2020:

  1. Unduh versi terbaru Autocad Architecture 2020 dari situs web Autodesk.
  2. Double klik pada file yang telah diunduh.
  3. Klik tombol "Mulai" dan ikuti petunjuk pemasangan di layar.
  4. Ketika proses instalasi selesai, klik tombol "Selesai".

Cara Memulai Autocad Architecture 2020

Berikut adalah langkah-langkah untuk memulai Autocad Architecture 2020:

  1. Klik Start > All Programs > Autocad Architecture 2020.
  2. Klik "New" untuk membuat file baru.
  3. Pilih jenis file yang ingin Anda buat.
  4. Masukkan nama file dan klik tombol "OK".
  5. Klik tombol "OK" untuk memulai desain Anda.

Cara Menggunakan Autocad Architecture 2020

Berikut adalah cara menggunakan Autocad Architecture 2020:

  1. Klik pada alat desain yang ingin Anda gunakan, misalnya alat penggaris atau alat segitiga.
  2. Tetapkan dimensi dan ukuran objek yang ingin Anda buat.
  3. Klik tombol "Draw" untuk memulai menggambar.
  4. Klik pada titik yang Anda inginkan untuk menarik garis.
  5. Klik pada titik yang berbeda untuk menarik garis lain.
  6. Klik tombol "OK" untuk menyelesaikan desain Anda.

Cara Membuat Desain Arsitektur dengan Autocad Architecture 2020

Berikut adalah cara membuat desain arsitektur dengan Autocad Architecture 2020:

  1. Pilih alat desain yang ingin Anda gunakan.
  2. Pilih objek yang ingin Anda buat, seperti bangunan, jendela, atap, dan lain-lain.
  3. Tetapkan dimensi dan ukuran objek yang ingin Anda buat.
  4. Klik tombol "Draw" untuk memulai menggambar.
  5. Klik pada titik yang Anda inginkan untuk menarik garis.
  6. Klik pada titik yang berbeda untuk menarik garis lain.
  7. Klik tombol "OK" untuk menyelesaikan desain Anda.

Cara Menyimpan Desain dengan Autocad Architecture 2020

Berikut adalah cara menyimpan desain dengan Autocad Architecture 2020:

  1. Klik File > Save As.
  2. Pilih jenis file yang ingin Anda simpan.
  3. Masukkan nama file dan klik tombol "Save".
  4. Klik tombol "Yes" untuk mengkonfirmasi penyimpanan desain Anda.

Kesimpulan

Autocad Architecture 2020 adalah aplikasi desain dan drafting yang kuat dan fleksibel yang dapat membantu Anda dalam menciptakan desain arsitektur yang Anda inginkan. Aplikasi ini memiliki berbagai fitur dan alat yang berguna untuk membantu Anda menciptakan desain yang kompleks dengan mudah. Dengan Autocad Architecture 2020, Anda dapat menciptakan desain arsitektur yang kompleks dan indah dengan mudah.
